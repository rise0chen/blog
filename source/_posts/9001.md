ubuntu下载软件慢的原因大部分是因为软件源地址是国外的，更新软件源，可以提升很大的下载速度。
<!--more-->

## 一、备份源列表
按 ctrl+alt+t 打开一个新的终端，输入：
```bash
sudo cp /etc/apt/sources.list /etc/apt/sources.list.bak
```

## 二、打开 并替换所有内容
```bash
sudo vi /etc/apt/sources.list
```

## 三、更新列表
```bash
sudo apt-get update
```
