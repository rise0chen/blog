## 一、连接MySQL
``` bash
mysql -h主机地址 -P端口 -u用户名 -p用户密码
```

## 二、备份MySQL
``` bash
mysqldump -h主机地址 -P端口 -u用户名 -p用户密码 --databases <db1 db2 db3> > ./mysql/db.sql
```

## 三、恢复MySQL
``` bash
mysql -h主机地址 -P端口 -u用户名 -p用户密码 < ./mysql/db.sql
```
