WeUI从v0.4.3到v1.0.0升级时，采用BEM命名规范。
由之前单一的下划线改成weui-aaa__bbb-bbb_ccc-ccc这种形式，横杠为连词符，双下划线区分Block和Element（BEM），单下划线连接Modify。
<!--more-->

| v0.4.3    | v1.0.0    |
| --------- | --------- |
| weui_     | weui-     |
| _hd       | __hd      |
| _bd       | __bd      |
| _ft       | __ft      |
| _ft       | __ft      |
| _title    | __title   |
| _area     | -area     |
