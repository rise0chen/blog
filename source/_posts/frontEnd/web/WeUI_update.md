WeUI 从 v0.4.3 到 v1.0.0 升级时，采用 BEM 命名规范。
由之前单一的下划线改成 weui-aaa\_\_bbb-bbb_ccc-ccc 这种形式，横杠为连词符，双下划线区分 Block 和 Element（BEM），单下划线连接 Modify。

<!--more-->

| v0.4.3  | v1.0.0    |
| ------- | --------- |
| weui\_  | weui-     |
| \_hd    | \_\_hd    |
| \_bd    | \_\_bd    |
| \_ft    | \_\_ft    |
| \_ft    | \_\_ft    |
| \_title | \_\_title |
| \_area  | -area     |
